Take payments on Bitcoins and other Cryptocurrencies using Compralo.

== Description ==

The **Compralo for WooCommerce** plugin allows retailers to accept payments via **cryptocurrencies** using the Compralo API. By completing a purchase users are redirected to the Compralo platform.

**Compralo** works with the major currencies in the market, so you have the autonomy to choose how you want to pay and receive. **Cryptocurrencies available:**

- Bitcoin
- Ethereum
- Mindexcoin
- Easticoin

== Installation ==

1. Access the WordPress admin panel.
2. In the right sidebar, go to: Plugins> Add New.
3. In the search field enter "Compralo for WooCommerce". 
4. Select our plugin from the list and click "Install Now".
5. Finally, click the "Activate" button after installation.

= Manual Installation =

The manual installation method involves downloading our plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here] (http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

== Requirements ==

- [Compralo](compralo.io) account.
- API access key.
- Plugin "WooCommerce" activated and installed.
- PHP 5.6 minimum version
